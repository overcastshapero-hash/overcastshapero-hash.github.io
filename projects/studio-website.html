Project · 网页设计

Studio

用 Cursor 构建的 AI 驱动网页设计工作室 Landing Page，探索玻璃态视觉语言与动态交互设计的边界。

这个项目的出发点

我是 UX 设计师出身，一直对"网页设计本身能不能成为作品"这件事很感兴趣。这个项目不是一个功能性产品，它更接近于一次视觉语言实验——用黑色、玻璃态、粒子动画构建一套高密度的设计系统。

全程用 Cursor 完成，从零开始写 HTML/CSS/JS，没有依赖任何 UI 框架。这让我对"AI 辅助写界面"这件事有了更直接的认识：Cursor 的优势不在于一键生成，而在于让我能专注在设计决策上，而不是在调 CSS 上耗时间。

设计系统核心
  • Liquid Glass 玻璃态：自定义双层玻璃效果——弱玻璃（blur 4px）与强玻璃（blur 50px）区分层级，内边框用渐变 mask 模拟光折射，不用图片，纯 CSS 实现
  • Canvas 粒子网格：Hero、Stats、CTA 三区域各有独立粒子系统，点间距离感应连线，粒子数量和速度按区域差异调整，形成空间感
  • 逐字动画标题：将 Hero 标题拆分为单字 span，通过 CSS animation-delay 形成级联入场，视觉重量聚焦
  • 数字滚动计数器：进入视口时触发 easeOutCubic 曲线动画，三段数字使用不同时长制造节奏感
用 Cursor 写界面的真实体验

这是我第一次完整地用 Cursor 从头写一个视觉驱动的页面。最大的收获不是"速度快了多少"，而是发现 AI 非常擅长帮我写我知道想要但不记得语法的东西——比如 Canvas 粒子系统、CSS mask 合成、IntersectionObserver 的使用方式。

设计判断本身没法外包。我知道我想要什么视觉结果，Cursor 帮我把这个意图翻译成代码。这个分工很清晰，也很舒服。

状态

第一版已完成。定位为视觉实验和设计语言研究，不作为线上服务发布。

技术构成
  • 纯 HTML / CSS / Vanilla JS
  • Instrument Serif + Barlow 字体组合
  • Canvas API 粒子系统（4 个独立实例）
  • CSS custom property 主题变量
  • IntersectionObserver 滚动触发
  • Cursor AI 辅助开发
设计决策记录
  • 黑底白字：强迫视觉焦点，排除干扰
  • 无框架：避免 Tailwind 的视觉均质化
  • Serif 大标题：在暗背景上制造质感反差
  • 粒子系统：让页面"呼吸"，弱化静态感
